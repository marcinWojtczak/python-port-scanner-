# Python Port Scanner

  Simple TCP port scanner written in Python for learning purposes.

  ## Features
  - Scan common ports (FTP, SSH, HTTP, HTTPS, RDP, MySQL, PostgreSQL)
  - Custom target IP/hostname
  - Service name detection

  ## Usage
  ```bash
  python3 port_scanner.py <target>
  python3 port_scanner.py 192.168.1.1
  python3 port_scanner.py scanme.nmap.org

  Educational Purpose

  Built as part of cybersecurity learning path. For authorized testing only.
